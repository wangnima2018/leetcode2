<h3>题意<h3>
<p>把一个字符串里面的每个单词，都reverse：
Given s = "the sky is blue",
return "blue is sky the".
<p>

<h3>解题<h3>
<p>把每个单词split出来，然后逐个进行reverse<p>

```python
class Solution(object):
    def reverseWords(self, s):
        """
        :type s: str
        :rtype: str
        """
        ssplit = s.split()
        res = ""
        for i in xrange(len(ssplit)-1,-1,-1):
            res += ssplit[i]
            res += " "
        res = res[:len(res)-1]
        return res
```
